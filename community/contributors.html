How to contribute

There are multiple ways you can contribute to the project. And help is always welcome!

Issue Tracker

Apache Yunikorn (Incubating) uses JIRA as an Issue Tracker. Don’t hesitate to report new bugs and improvements ideas.

Contribute to the source code

We like to get code contributions through Pull Requests on our variouos repos Yunikorn Core, Yunikorn K8s Shim, Yunikorn Web, Yunikorn Scheduler Interface.

But before starting, please read our Contribution guidelines, it will give you important information about our review process, and pointers on how to make a good code contribution.

You can visit our Issue Tracker to find issues to resolve.

Other contributions

Not much of a coder? There are other ways to help out:

  • Documentation and website improvements are always welcome
  • Helping each other by answering questions on the Mailing List
  • Participating in reviewing contributions.

Mailing list

Get help using Apache Yunikorn (Incubating) or contribute to the project on our mailing lists:

  • Users : subscribe, unsubscribe, archives
    for usage questions, help, and announcements.
  • Dev : subscribe, unsubscribe, archives
    for people wanting to contribute to the project.
  • Commits : subscribe, unsubscribe, archives
    for commit messages and patches.
